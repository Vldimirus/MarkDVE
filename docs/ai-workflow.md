# Сценарий работы с AI через Markdown

Первая версия не запускает AI-ассистента из интерфейса напрямую, но уже готова к работе с внешними Markdown-файлами.

## Базовый поток

1. AI-ассистент генерирует или обновляет `.md` файлы в рабочей папке проекта.
2. Разработчик открывает эти документы в MarkDVE как карту проекта.
3. Из главного документа можно переходить к отдельным модулям, экранам, БД и планам.
4. При изменении файла на диске открытая вкладка может быть перечитана и обновлена.

## Что можно добавить дальше

- Панель истории изменений Markdown-карты.
