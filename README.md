# MarkDVE

Russian version: [README.ru.md](README.ru.md)

MarkDVE is a desktop Markdown viewer and editor built with C++, Qt 6 Widgets, and CMake.

The project is designed for working with project documentation in Markdown format:
- viewing Markdown files in tabs;
- navigating through links inside Markdown documents;
- opening the source `.md` file in a separate editor window;
- configuring the appearance of both the viewer and the editor;
- keeping a history of recently opened documents;
- generating a cross-linked Markdown code map from a selected source directory.

The application is aimed at project knowledge navigation, technical documentation browsing, and structured code map generation directly from source files.
