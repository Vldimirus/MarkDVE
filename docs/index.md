# MarkDVE

MarkDVE — настольный просмотрщик и редактор Markdown-документов для ведения карты проекта.

## Основные разделы

- [Глобальная карта проекта](Project_Map/map_main.md)
- [Просмотрщик Markdown](modules/viewer.md)
- [Отдельное окно редактора](modules/editor.md)
- [Рабочая папка проекта](modules/workspace.md)
- [Сценарий работы с AI через Markdown](ai-workflow.md)

## Первый сценарий работы

1. Открыть рабочую папку проекта.
2. Выбрать нужный `.md` в дереве документов.
3. Перейти по ссылкам между страницами карты проекта.
4. Открыть текущую страницу в отдельном окне редактора.
5. Сохранить изменения и сразу увидеть обновлённый просмотр.
