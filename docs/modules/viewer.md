# Просмотрщик Markdown

Просмотрщик отображает Markdown-файлы во вкладках и позволяет переходить по ссылкам между документами.

## Что уже сделано

- Каждый открытый документ появляется в отдельной вкладке.
- Повторное открытие существующего документа активирует уже открытую вкладку.
- Ссылки на Markdown-файлы внутри рабочей папки открываются прямо в приложении.
- Внешние ссылки можно передать системному обработчику.
- Последние открытые Markdown-файлы сохраняются в отдельном меню.
- Есть отдельное окно настройки внешнего вида просмотра с живым примером в редакторе и в просмотре.

## Следующие шаги

- Добавить подсветку текущего раздела в дереве документов.
- Расширить поддержку якорей и оглавлений.
