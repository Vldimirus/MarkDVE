# Отдельное окно редактора

Редактор открывает исходный `.md` файл отдельно от окна просмотра.

## Возможности

- Открытие текущего документа из главного окна.
- Сохранение в UTF-8.
- Перечитывание файла с диска.
- Предупреждение о несохранённых изменениях.
- Настройка шрифта и табуляции.
- Чекбоксы включения типов Markdown-подсветки.
- Подсветка заголовков, ссылок, цитат, списков, чекбоксов и кода.

## Идея развития

На следующих этапах сюда можно добавить split-view, предпросмотр diff, настройку цветов подсветки и шаблоны для проектных карточек.
