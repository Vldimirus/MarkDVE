# MarkDVE

English version: [README.md](README.md)

MarkDVE — это настольный просмотрщик и редактор Markdown, построенный на C++, Qt 6 Widgets и CMake.

Проект предназначен для работы с документацией проекта в формате Markdown:
- просмотр Markdown-файлов во вкладках;
- переход по ссылкам внутри Markdown-документов;
- открытие исходного `.md` файла в отдельном окне редактора;
- настройка внешнего вида просмотра и редактора;
- хранение истории недавно открытых документов;
- генерация перекрёстно связанной Markdown-карты кода по выбранной папке исходников.

Приложение ориентировано на навигацию по знаниям о проекте, просмотр технической документации и генерацию структурированной карты кода напрямую из исходных файлов.
